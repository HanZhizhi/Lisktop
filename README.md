# Lisktop
## 关于LiskTop

​	LiskTop以为_Desktop with list_，即包含提醒事项的桌面启动器，受到谷歌“Desert Island" 这一digital wellbeing套件中的桌面启动器启发，基于其功能之上添加效能功能：

1. 包含主界面和应用列表页，高效切换；
2. 应用列表可根据Lru、冒泡法、限制应用等多种方式进行排序，专注于最快打开所需应用；
3. 主界面包含待办事项列表，可自由修改；
4. 更多辅助功能，如定时打开应用等；
